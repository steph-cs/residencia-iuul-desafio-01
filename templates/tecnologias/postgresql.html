PostgreSQL

O que é PostgreSQL ?

O PostgreSQL é um sistema avançado de gerenciamento de banco de dados objeto-relacional que oferece suporte a um subconjunto estendido do padrão SQL, incluindo transações, chaves estrangeiras, subconsultas, gatilhos, tipos definidos pelo usuário e funções.

PostgreSQL é uma ferramenta de código aberto com 11,4 mil estrelas e 3,6 mil forks no GitHub.

GitHub Site oficial

Vantagens

  • Código aberto
  • Altamente expansível
  • Possibilidade de processar tipos de dados complexos (por exemplo, dados geográficos)
  • Bom suporte a linguagens
  • Multiplataforma

Desvantagens

  • Não disponível em todos os hosts por padrão
  • Documentação expansível disponível apenas em inglês
  • Velocidade de leitura comparativamente baixa
