CSS 3

O que é CSS 3 ?

CSS (Cascading Style Sheets) é definido como uma linguagem de folha de método que fornece aos web designers controle sobre como um site da Internet se comunica com navegadores da Web, incluindo a formatação e exibição de seus documentos HTML.

CSS pode ser uma linguagem de codificação baseada em texto que especifica os formatos do site e a maneira como um site se comunica com os navegadores da Web. A linguagem permite que os desenvolvedores da Web regulem vários elementos e funcionalidades de estilo, como layout, cor, fontes e, portanto, a formatação e exibição de documentos HTML.

Site oficial

Vantagens

  • Especificar um estilo repetido para o elemento uma vez e usá-lo várias vezes
  • Estilo é aplicado de forma consistente
  • Simplifica o desenvolvimento do site e manutenção
  • Reduz o tamanho da transferência de arquivos

Desvantagens

  • Compatibilidade do navegador
  • Escassez de segurança
  • Existem vários níveis, o que cria confusão para não-desenvolvedores e iniciantes
