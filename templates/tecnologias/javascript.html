JavaScript

O que é JavaScript ?

JavaScript é mais conhecido como a linguagem de script para páginas da Web, mas usado em muitos ambientes não-navegador, bem como node.js ou Apache CouchDB. É uma linguagem de script multiparadigma baseada em protótipos que é dinâmica e suporta estilos de programação orientados a objetos, imperativos e funcionais.

Site oficial

Vantagens

  • É simples de compreender e aprender
  • É uma linguagem "interpretada", o que reduz o tempo necessário para a compilação
  • Se integra perfeitamente a outras linguagens de programação
  • Validação de dados pode ser feita dentro do próprio navegador

Desvantagens

  • Não é possível depurar
  • Parada inesperada da renderização
  • Segurança do lado do cliente
  • Não suporta herança múltipla
  • Não suporta navegadores antigos
