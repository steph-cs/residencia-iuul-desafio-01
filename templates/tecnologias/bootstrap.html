Bootstrap

O que é Bootstrap ?

O Bootstrap é o framework HTML, CSS e JS mais popular para o desenvolvimento de projetos responsivos e móveis na Web.

O Bootstrap é uma ferramenta de código aberto com 160,5 mil estrelas e 77,6 mil forks no GitHub.

GitHub Site oficial

Vantagens

  • É responsivo
  • Reajusta automaticamente imagens e elementos
  • Construído com JavaScript e CSS
  • Facilmente customizavél
  • Muitos templates

Desvantagens

  • Requer que você saiba como codificar
  • Seu esquema de nomenclatura pode ser confuso
  • Diferentes suportes de vídeo para diferentes navegadores
  • Pode ser menos simplificado do que você deseja
  • Você pode se tornar muito dependente dele
