Java

O que é Java ?

Java é uma linguagem de programação e plataforma de computação lançada pela Sun Microsystems em 1995. Existem muitos aplicativos e sites que não funcionarão a menos que você tenha o Java instalado, e mais são criados todos os dias. Java é rápido, seguro e confiável. De laptops a datacenters, consoles de jogos a supercomputadores científicos, telefones celulares à Internet, Java está em toda parte!

Site oficial

Vantagens

  • Sintaxe do Java é direta, fácil de escrever, aprender, manter e entender
  • Linguagem de programação orientada a objetos
  • Reduz as ameaças e os riscos de segurança
  • Barato e econômico de manter
  • Independente de plataforma

Desvantagens

  • Consome memória e é significativamente mais lento do que linguagens nativas
  • Fornece aparência não tão atraente da GUI
  • Não fornece nenhum recurso de backup
  • Requer espaço de memória significativo
  • Códigos detalhados e complexos
