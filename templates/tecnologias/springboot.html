O que é SpringBoot ?

HTML5 é uma linguagem de marcação de tecnologia central da Internet usada para estruturar e apresentar conteúdo para a World Wide Web. A partir de outubro de 2014, esta é a quinta revisão final e completa do padrão HTML do World Wide Web Consortium (W3C).

Vantagens

  • Suporte multimídia
  • Sintaxe curta e simples
  • Recursos de segurança aprimorados
  • Incluir marcas semânticas
  • Suporte entre plataformas

Desvantagens

  • Renderização do lado do cliente
  • O armazenamento local é menos seguro
  • Diferentes suportes de vídeo para diferentes navegadores
  • Custo de licenciamento de mídia
  • Não suporta navegadores antigos
