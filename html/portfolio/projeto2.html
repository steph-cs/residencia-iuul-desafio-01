LokaCar

Descrição:

LokaCar é um sistema web de locação de veículos.

Desenvolvido para a disciplina de Programação para Web, onde foi desenvolvido conhecimento e habilidades para construção de Front e Back end.

O sistema permite aos usuários publicarem seu veículo (carro, moto, van) para locação e também buscar por um de acordo com sua localização para realizar a locação.

Para alugar ou publicar um veículo é necessário realizar cadastro/login.

É possível vizualizar e editar as informações do perfil do usuário, assim como seus anúncios publicados e reservas de veículo realizadas.
